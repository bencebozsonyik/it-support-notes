# Network Basics

## No internet connection but Wi-Fi connected
**Symptoms**
- Connected to Wi-Fi
- Websites do not load

**Troubleshooting**
1. Check IP address with `ipconfig`
2. Ping a website to test connectivity
3. Renew DHCP lease if needed

**Resolution**
Incorrect IP address assignment fixed by renewing DHCP.

---

## DNS issue
**Symptoms**
- Websites do not load by domain name
- Accessible via IP address

**Troubleshooting**
1. Check DNS server settings
2. Use `nslookup` to test resolution
3. Restart router/modem

**Resolution**
Updating DNS settings resolved the problem.
