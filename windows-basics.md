# Windows Basics

## User cannot log in to Windows
**Symptoms**
- Incorrect password message
- Caps Lock enabled

**Troubleshooting**
1. Check keyboard layout
2. Check Caps Lock status
3. Reset password via admin account if needed

**Resolution**
User successfully logged in.

**Notes**
Always check simple input issues (keyboard layout, Caps Lock, Num Lock) before more advanced fixes.

---

## Slow system
**Symptoms**
- PC responds slowly
- Programs open with delay

**Troubleshooting**
1. Check running programs (Task Manager)
2. Check background processes
3. Restart the computer

**Resolution**
Closing unnecessary programs and restarting improved system performance.

**Notes**
Keep Windows and drivers updated regularly.
